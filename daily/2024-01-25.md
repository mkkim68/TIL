# 백준 24313번 [알고리즘 수업 - 점근적 표기 1](https://www.acmicpc.net/problem/24313)
```python
import sys  
  
a1, a0 = map(int, sys.stdin.readline().split())  
c = int(sys.stdin.readline())  
n0 = int(sys.stdin.readline())  
  
n = n0  
  
for i in range(3):  
    f = a1 * n + a0  
    g = c * n  
    if f > g:  
        print(0)  
        break  
    else:  
        pass
```
