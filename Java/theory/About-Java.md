# 자바는 어떤 언어일까?
## 컴파일 언어
- 코드를 미리 compile해 컴퓨터가 읽을 수 있게 함
	- 하지만 운영체제에 따라 컴파일 방식이 다름!
	- 이를 위해 JVM 도입
## JVM (Java Virtual Machine)
- 두 가지 언어를 할 줄 앎
	- 운영체제에 맞는 언어
	- JB (Java Bitecode): 자바로 짠 코드가 컴파일 되는 언어
## JRE (Java Runtime Environment)
- 자바 실행 환경
- 운영체제 내에서 JVM이 속한 환경
	- java로 짠 코드를 실행하기 위해 컴퓨터에 설치되어야 함
- 포함하는 것
	- JVM
	- 표준 라이브러리
	- 등
- 예전엔 실행만 할 컴퓨터에 다운받을 수 있도록 JRE 따로 제공 (현재는 X)
## JDK (Java Development Kit)
- 개발자가 자바로 프로그래밍하는 전 과정을 도와주는 자바 코드 제작 키트
- 포함하는 것
	- JRE
	- 컴파일러
	- 디버거
	- JAR 도구
	- 프로파일러
	- 등
