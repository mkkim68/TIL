# 주석
## 방법
### 한 줄
```java
//  한 줄짜리 주석  
System.out.println("오른쪽을 봐"); // 주석이 요기잉네?  
```
### 여러 줄
```java
/* 주석은 이와 같이 여러 줄에 걸쳐  
작성할 수도 있습니다.  
장문의 주석을 달거나  
  
  
System.out.println("아...!");  
System.out.println("안돼!");  
  
위와 같이  
코드의 일부를 주석처리하여  
비활성화하는 등의 목적으로 사용되죠. */
```

## 단축키
- `command + /`
- 중간에 원하는 부분만 주석처리/해제 가능
## 기타
### 주석의 양은 실행속도에 영향을 주지 않음
- 컴파일 과정에서 배제되기 때문
### 주석은 가능한 많이 달지 않는 것이 좋음
- 주석이 많이 필요한 코드: 나쁜 코드
- 주석이 필요하지 않도록 짜야 함
