# 정수 자료형들과 관련 연산자
## 정수 자료형
| 자료형   | 크기           | 표현 범위                                                  |
| ----- | ------------ | ------------------------------------------------------ |
| byte  | 1byte (8bit) | -128 ~ 127 (-2^7 ~ 2^7)                                |
| short | 2byte        | -32,768 ~ 32,767                                       |
| int   | 4byte        | -2,147,483,648 ~ 2,147,483,647                         |
| long  | 8byte        | -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807 |
```java
byte _1b_byte = 1;
short _2b_short = 2;
int _4b_int = 3;
long _8b_long = 4;
```
- 자료형의 범주 외의 수를 담을 수 없음
- 큰 자료형에 작은 자료형의 값을 넣을 수 있음
	- [I] 묵시적(암시적) 형변환
- [!] 작은 자료형에 큰 자료형의 값을 '그냥' 넣을 수 없음
	- 들어있는 값의 크기와 무관
- ⭐int의 범위를 벗어나는 수에는 리터럴에도 명시 필요
	- 끝에 `l` 또는 `L` 을 붙여볼 것
```java
long _8b_long1 = 123456789123456789L;  
```
- 가독성을 위해 아래와 같이 표현 가능
```java
int _4b_int1 = 123_456_789;
long _8b_long2 = 123_456_789_123_456_789L;
```
### 형변환 (casting)
- 명시적(강제) 형변환
- 개발자: "내가 책임질테니까 그냥 넣으세요."
```java
byte byteNum;  
int smallIntNum = 123;  
  
byteNum = (byte) smallIntNum;
```
- [!] 강제로 범주 외의 값을 넣을 경우 값 손실
```java
int intNum = 12345;  
byteNum = (byte) intNum; // byteNum: 57
```
- [I] `12345 % 128` 연산 수행하고 억지로 넣음
### 이항연산자
- `+`, `-`, `*`, `/`, `%`
- 숫자 데이터(리터럴)와 변수 혼합 사용 가능
- 다중으로 사용 가능, 단 연산자별 우선순위 있음
	- 괄호로 해결
- 부수효과 없음 확인
#### byte, short 연산
```java
byte b1 = 1;  
byte b2 = 2;  
short s1 = 1;  
short s2 = 2;  
  
// 전부 불가능
byte b3 = b1 + b2;  
short s3 = b1 + b2;  
short s4 = b1 + s2;  
short s5 = s1 + s2;
```
- ⭐byte와 short의 연산들은 int 반환
	- 더 작은 수에도 그냥 int를 많이 쓰는 이유 중 하나
```java
// 가능
int i1 = b1 + b2;  
int i2 = s1 + s2;  
int i3 = b1 + s2;
```
- 메모리를 크게 절약해야 하는 상황이 아닌 이상 `int` 널리 사용
##### int를 널리 사용하는 기타 이유들
- java 및 다수 언어들에서 기본 자료형으로 지정됨
	- 라이브러리와 API 등에서 사용
- 다른 언어들과 호환 (널리 사용되는 자료형)
- 연산 속도가 타 자료형보다 빠름
	- 32비트(4바이트): 대부분의 **CPU에서 처리하기 적합한 크기**
#### long 연산
```java
long l1 = 1;  
long l2 = 2;  
  
int l3 = l1 + l2; // Error
```
- long끼리의 연산은 long 반환
#### 소수점 아래 **버림**
```java
byte int1 = 5/2;  // 2
int int2 = 10;  
int int3 = 3;  
int int4 = int2/int3;  // 3
```
### 복합 대입 연산자
| `a += b` | `a = a + b` |
| -------- | ----------- |
| `a -= b` | `a = a - b` |
| `a *= b` | `a = a * b` |
| `a /= b` | `a = a / b` |
| `a %= b` | `a = a % b` |
```java
int a = 1;  
a = a + 2;  // a: 3
  
a += 3;  // a: 6

// 값을 반환하기도 함
int b = a += 4; // a: 10, b: 10
```
#### 추가
- [I] 일반 대입 연산자도 값을 반환함
	- 괄호 있으나 없으나 같음
```java
int i1 = 0;  
int i2 = 1;  
int i3 = i1 = i2;  // i1: 1, i2: 1, i3: 1
  
String s1 = "ABC";  
String s2 = "가나다";   
String s3 = (s1 = s2);  // s1: 가나다, s2: 가나다, s3: 가나다
```
#### 자료형의 범위를 넘어가는 연산
- 자료형의 범위를 넘어가도록 숫자를 더하거나 뺄 경우
```java
byte x = 127;  
x += 1;  // x: -128
  
byte y = -128;  
y -= 1;  // y: 127
```
### 단항 연산자
| 연산자   | 반환값     | 부수효과 |
| ----- | ------- | ---- |
| `+`   | 값 그대로   | 없음   |
| `-`   | 양음 반전   | 없음   |
| x`++` | 값 그대로   | 1 증가 |
| `++`x | 1 증가한 값 | 1 증가 |
| x`--` | 값 그대로   | 1 감소 |
| `--`x | 1 감소한 값 | 1 감소 |
```java
int int1 = 3;  // int1: 3
  
int int2 = int1++;  // int2: 3, int1: 4
int int3 = ++int1;  // int3: 5, int1: 5
int int4 = -(int2-- * --int3); // int2: 3, int3: 4, int4: -12
// int2: 2
```
#### 리터럴에는 사용 불가
```java
// 불가능
int int5 = 3++;  
int int6 = --3;
```
### 비교 연산자
| `a == b`  | a와 b는 같다      |
| --------- | ------------- |
| `a ! = b` | a와 b는 다르다     |
| `a > b`   | a가 b보다 크다     |
| `a >= b`  | a가 b보다 크거나 같다 |
| `a < b`   | a가 b보다 작다     |
| `a <= b`  | a가 b보다 작거나 같다 |
- `boolean` 자료형의 값을 반환
#### 다른 정수 자료형끼리 사용 가능
```java
int int1 = 3;  
int int2 = 1;  
  
boolean bool1 = int1 == int2;  
boolean bool2 = int1 != int2;  
```
#### 우선순위에 따른 연산
```java
int int1 = 3;  
int int2 = 1;  
  
boolean bool7 = int1 * int2 > int1 + int2;
```
