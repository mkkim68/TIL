# 비트 연산자
## 특징
- [!] 단축평가가 일어나지 않음
## 종류
### and
- `&`
```java
int x = 5;  // 0 0 1 0 1
int y = 19; // 1 0 0 1 1

int x_and_y = 5 & 19;  // 0 0 0 0 1
```
### or
- `|`
```java
int x = 5;  // 0 0 1 0 1
int y = 19; // 1 0 0 1 1

int x_or_y = 5 | 19;  // 1 0 1 1 1
```
### xor
- `^`
- 다르면 1, 같으면 0
```java
int x = 5;  // 0 0 1 0 1
int y = 19; // 1 0 0 1 1

int x_xor_y = 5 ^ 19;  // 1 0 1 1 0
```
### not
- `~`
```java
int x = 5;  // 0 0 1 0 1
int y = 19; // 1 0 0 1 1

int notX = ~5;  // 1 1 0 1 0
int notY = ~19;  // 0 1 1 0 0
```
### shift
- `<<`, `>>`
```java
int x = 5;  // 0 0 1 0 1
int x_L_1 = 5 << 1;  // 0 1 0 1 0

int y = 19; // 1 0 0 1 1
int y_R_1 = 19 >> 1; // 0 1 0 0 1
```
