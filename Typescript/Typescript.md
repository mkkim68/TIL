# Typescript란?
- JS에 타입을 부여한 언어
	- JS의 확장된 언어(superset)
	- JS 기능 + 더 많은 기능
- 컴파일해서 js로 변환한 후 사용
- 정적 타입: 컴파일 타임 코드 작성 단계에서 오류 확인
## 특징
- 개발 환경에서 에러를 잡는 걸 도와줌
- type annotations를 사용해서 코드 분석 가능
- 오직 개발 환경에서만 활성화
- 성능 향상과는 관계 없음
## 목적
- JS 코드를 단순화하여 더 쉽게 읽고 디버그할 수 있도록 함
- 코드 유형 검사를 통해 JS를 작성할 때 개발자가 일반적으로 겪는 버그를 피하는데 도움이 됨
